# 로또 게임

## 기능 구현 목록

1. 사용자 인터페이스 구현
- [x] 로또 구입 금액을 입력한다.
- [x] 지난 주 당첨 번호와 보너스 볼을 입력한다.
2. 입력값 유효성 검사
- [x] 0미만 정수 불가 
- [x] 구입금액은 1000의 배수여야 한다.
- [x] 공백 값 및 "," 입력 불가능
- [x] 100000원 초과 구매 불가 
- [x] 보너스볼 값 범위 1~45여야 한다.
3. 난수 생성
- [x] 1~45 사이 임의의 값을 6개 생성한다. 
4. 게임 진행
- [x] 구입한 로또 갯수와 찍은 로또 숫자를 출력한다.
- [ ] 당첨 통계를 출력한다.
- [ ] 수익률을 계산한다.
